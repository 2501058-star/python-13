# 파이썬 프로그래밍 실습 (13주차 과제)

**과제 범위:** `파이썬실습_#7.pdf` (p.37) 과제 #39 ~ #45
**주제:** 함수(Function) 정의 및 활용

---

### 과제 풀이 내용

**과제 #39 (홀수 출력)**
- 사용자 정의 함수(`def`)를 선언하고, `for`문과 `if`문(`% 2 != 0`)을 사용해 범위 내 홀수만 출력하도록 작성했습니다.

**과제 #40 (3의 배수 출력)**
- 숫자를 매개변수로 받아 3의 배수인지 확인하는 조건문을 함수 안에 구현했습니다.

**과제 #41 (최댓값, 최솟값 반환)**
- 4개의 숫자를 입력받고, `return max_x, min_x` 처럼 **한 번에 여러 개의 값을 반환(튜플 형태)**하는 방식을 사용했습니다.

**과제 #42 (홀수 출력 - 39번과 동일)**
- 문제 내용이 39번과 동일하여, 39번에서 만든 함수를 재사용하여 풀이했습니다.

**과제 #43 (팩토리얼 계산)**
- 1부터 입력받은 수까지 반복해서 곱하는 로직을 작성하고, 최종 결과값을 `return`으로 반환했습니다.

**과제 #44 (조건에 맞는 곱의 합계)**
- 이중 `for`문을 사용하여 두 수의 곱이 30 이상인 경우만 찾아 총합을 계산했습니다.

**과제 #45 (리스트 누적 합)**
- 리스트(`list`) 자체를 함수의 인수로 전달받아, 내부 요소들을 반복문으로 더해 결과를 출력했습니다.
